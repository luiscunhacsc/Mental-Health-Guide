# Guia de Saúde Mental (Aplicação Educativa)

## O que é esta aplicação?
Esta aplicação foi criada para apoiar **aprendizagem em Saúde Mental** de forma prática, clara e estruturada.

Foi pensada para:
- Estudantes de Psicologia
- Psicólogos em formação
- Profissionais que queiram rever sinais clínicos, estratégias de intervenção e raciocínio diagnóstico

## Para que serve
A aplicação ajuda a:
- Rever conceitos essenciais (definições e categorias clínicas)
- Estudar sinais e sintomas por área clínica
- Explorar estratégias de intervenção
- Treinar raciocínio clínico com **casos clínicos**
- Consolidar aprendizagem com **cartões** e **quiz**

## Como usar (sem instalação)
1. Abra o ficheiro `index.html`.
2. Navegue pelos módulos no menu lateral.
3. Use os modos `Conteúdo`, `Cartões` e `Quiz`.
4. No **Laboratório de Casos Clínicos**, selecione um caso e valide a sua hipótese diagnóstica.

## Percurso de estudo recomendado
1. **Definições Essenciais**
2. **Categorias de Doença Mental**
3. **Guias Clínicos** (sinais/sintomas + estratégias)
4. **Laboratório de Casos Clínicos**
5. **Cartões** para revisão rápida
6. **Quiz** para autoavaliação

## O que vai encontrar no Laboratório de Casos
- Historial resumido do caso
- Pistas clínicas organizadas
- Escolha da hipótese diagnóstica principal
- Feedback com justificação clínica
- Diagnóstico diferencial
- Ligação ao módulo teórico correspondente

## Nota importante (ética e prática clínica)
Esta aplicação é **educativa**.  
**Não substitui avaliação clínica formal, supervisão, consulta especializada, nem protocolos institucionais.**

## Privacidade
- A aplicação funciona localmente no navegador.
- O progresso de estudo é guardado apenas no próprio navegador (localmente).
